CS Resources

A curated list of Computer Science resources.

About

These are resources I've come across in my journey that have been either personally helpful to me (marked with a ⭐) or that people I trust have shared. I've also answered several frequently asked questions below.

Resources

Basic Programming

  • YouTube Playlist: Basic Python Programming by Sentdex⭐ This is a great playlist for learning the basics of Python and Programming in general.
  • Udemy: Java - The Complete Java Developer Course⭐ This is an excellent course for learning and getting good at Java and Object Oriented Programming.
  • The Missing Semester Of Your CS Education Covers everything from Vim to debugging to Git. Essential for a complete CS education.

Data Structures and Algorithms

  • cses.fi⭐ The CSES problem set is highly curated and covers a wide range of topics, the problems can get quite tough so don't be discouraged!
  • The Competitive Programmer's Handbook⭐ This is a great book to get a crash course in the basics of data structures and algorithms, think of it as a companion piece for the above problemset.
  • USACO Guide I used this guide when I was tutoring people in Competitive Programming, it's an exceptional resource in learning competitive programming and will prepare you more than adequately for DSA interviews.
  • Introduction to Algorithms, 3rd Edition by CLRS⭐ This is a mathematically rigorous textbook that covers data structures and algorithms in depth. Instead of reading it front to back, use it as reference material for when you need a deep understanding of a particular concept. Excellent problems. If you don't want to buy it, I'm sure you can find a pdf of it online.

Machine Learning

  • YouTube Playlist: Machine Learning by Sentdex Think of this as a sequel to the basic Python programming playlist. Great intro to Machine Learning.
  • Fast.ai Course⭐ I followed an older version of this course to learn the basics of deep learning for the first time, all you really need to know is Python, it's great!!!
  • Seeing Theory Fun way of gaining intuition for probability and statistics concepts, very important to know to have intuition for ML.
  • MIT OCW: Linear Algebra Excellent online course for learning Linear Algebra, if you don't know Linear Algebra, you're kinda cooked, so take the course.

Web Development

  • The Odin Project I learned web dev via random projects and online tutorials, but this lays out a roadmap and teaches you the essentials in a very structured manner. Still make sure to build projects in your own time.

Miscellaneous

  • Designing Data Intensive Applications⭐ I feel like I learned a good bit about System Design reading this, also people much more experienced than me recommend it.
  • Teach Yourself CS This is the inspiration for this page, and contains several more resources for a bunch of different topics traditionally covered in a CS degree, go check it out.

Advice / FAQs

How do I get better at programming?

Just write more code lol, all you can do is practice. I'm slightly above mid, but I get better by writing more code.

How do I come up with side project ideas?

Do you have inconveniences in your life? If so, try to solve them with software. Alternatively, go through this github repository

How do I get started?

Just learn a programming language, any language, with the intention of building a specific project or idea no matter how simple you think it seems.
